Bab 1: Algoritma ‚Äì Kerangka Berpikir Komputer

Algoritma adalah urutan langkah logis untuk menyelesaikan masalah. Ibarat resep masakan: tanpa resep, koki bisa salah. Tanpa algoritma, program bisa kacau.

Contoh algoritma menghitung luas persegi panjang:

1. Mulai
2. Masukkan panjang (P)
3. Masukkan lebar (L)
4. Hitung luas = P * L
5. Tampilkan luas
6. Selesai

Nanti kita akan terjemahkan ke Python. Fokus kita di modul ini adalah pemrograman Python, tapi selalu ingat algoritma di belakangnya.

Soal 1.1

Buat algoritma untuk:

  1. Mencari bilangan terbesar dari tiga angka.
  2. Menghitung rata-rata tiga nilai ujian.
  3. Menentukan apakah suatu tahun adalah tahun kabisat.

Jawaban:

1. Algoritma mencari bilangan terbesar dari tiga angka:
1. Mulai
2. Masukkan a, b, c
3. Jika a > b dan a > c maka terbesar = a
4. Jika b > a dan b > c maka terbesar = b
5. Jika c > a dan c > b maka terbesar = c
6. Tampilkan terbesar
7. Selesai

2. Algoritma rata-rata tiga nilai:
1. Mulai
2. Masukkan nilai1, nilai2, nilai3
3. jumlah = nilai1 + nilai2 + nilai3
4. rata = jumlah / 3
5. Tampilkan rata
6. Selesai

3. Algoritma tahun kabisat:
1. Mulai
2. Masukkan tahun
3. Jika tahun % 400 == 0 maka kabisat = benar
4. Jika tahun % 100 == 0 maka kabisat = salah
5. Jika tahun % 4 == 0 maka kabisat = benar
6. Jika tidak, kabisat = salah
7. Tampilkan "kabisat" atau "bukan"
8. Selesai

2.1 Variabel dan Tipe Data ‚Äì Kotak Penyimpanan

Variabel adalah nama yang mewakili suatu nilai di memori. Python bersifat dynamically typed: kita tidak perlu menyebutkan tipe data, Python akan menentukannya dari nilai yang diberikan.

Contoh 1: Membuat variabel

nama = "Dewi"           # string (teks)
usia = 17               # integer (bilangan bulat)
tinggi = 162.5          # float (bilangan desimal)
menikah = False         # boolean (True/False)

print(nama)
print(usia)
print(tinggi)
print(menikah)

Apa yang terjadi jika kita mengganti nilai? Variabel bisa diubah kapan saja, bahkan dengan tipe berbeda. Tapi hati-hati, bisa membingungkan.

usia = 17
usia = "tujuh belas"    # sekarang usia menjadi string (tidak disarankan)

print(type(nama))   #

Konversi Tipe Data (Casting)

Kadang kita perlu mengubah tipe, misal dari string ke integer agar bisa dihitung.

angka_str = "100"

2.2 Input dan Output ‚Äì Berkomunikasi dengan Pengguna

Output: menggunakan print() untuk menampilkan teks atau nilai ke layar.

Input: menggunakan input() untuk meminta pengguna mengetik sesuatu. Hasil input() selalu bertipe string.

Contoh 2: Program sapa sederhana

Bagaimana jika ingin input angka? Harus dikonversi.

print("Tahun depan usia Anda", usia + 1)

Format Output yang Rapi ‚Äì f-string dan {}

f-string (format string) adalah cara modern untuk menyisipkan variabel ke dalam string. Tanda {} digunakan sebagai tempat variabel atau ekspresi.

Penjelasan: Huruf f sebelum petik menandakan f-string. Di dalam {} kita bisa menempatkan variabel atau ekspresi seperti {umur+1}.

2.3.3 Operator Logika (and, or, not)

Digunakan untuk menggabungkan kondisi boolean.

  • not : membalikkan nilai (True jadi False, sebaliknya).

print(not a)     # False

2.3.4 Operator Penugasan (Assignment)

Selain =, ada operator gabungan seperti +=, -=, dll. Ini adalah cara singkat untuk mengubah nilai variabel.

c = 10
c += 5   # sama dengan c = c + 5 -> 15
c -= 3   # c = c - 3 -> 12
c *= 2   # 24
c /= 4   # 6.0 (float)
print(c)

2.3.5 Eksplorasi: Apa yang Terjadi Jika Operator Diganti?

Pertanyaan: Mengapa ada dua jenis pembagian (/ dan //)?

Jawab: / selalu menghasilkan float (desimal), sedangkan // menghasilkan pembulatan ke bawah (integer). Pilih sesuai kebutuhan: jika ingin hasil presisi, gunakan /; jika ingin hasil bulat (misal untuk indeks), gunakan //.

Pertanyaan: Apa bedanya = dan ==?

Jawab: = adalah penugasan (assignment), misal x = 5. == adalah perbandingan kesamaan. Salah satu kesalahan umum pemula adalah menulis if x = 5 (error) seharusnya if x == 5.

Bab 3: Percabangan ‚Äì Membuat Keputusan

Dengan if, elif, else, program dapat memilih tindakan berdasarkan kondisi.

3.1 If Tunggal

Contoh: Cek apakah suatu bilangan positif.

angka = int(input("Masukkan bilangan: "))
if angka > 0:
    print("Bilangan positif")

Jika angka <=0, tidak ada output.

3.2 If-Else ‚Äì Dua Cabang

angka = int(input("Masukkan bilangan: "))
if angka % 2 == 0:
    print("Genap")
else:
    print("Ganjil")

Mengapa pakai else? Karena bilangan pasti genap atau ganjil. Else mencakup semua kemungkinan yang tidak memenuhi if.

3.3 If-Elif-Else ‚Äì Banyak Kemungkinan

nilai = int(input("Masukkan nilai: "))
if nilai >= 90:
    grade = "A"
elif nilai >= 80:
    grade = "B"
elif nilai >= 70:
    grade = "C"
elif nilai >= 60:
    grade = "D"
else:
    grade = "E"
print(f"Grade: {grade}")

Perhatikan urutan: dari yang paling tinggi ke rendah. Jika dibalik, logikanya salah.

3.4 Nested If (If di dalam If)

Contoh: Menentukan jenis segitiga.

a = float(input("Sisi a: "))
b = float(input("Sisi b: "))
c = float(input("Sisi c: "))
if a + b > c and a + c > b and b + c > a:
    if a == b == c:
        print("Segitiga sama sisi")
    elif a == b or b == c or a == c:
        print("Segitiga sama kaki")
    else:
        print("Segitiga sembarang")
else:
    print("Bukan segitiga")

Pertama cek syarat segitiga, baru kemudian jenisnya.

Bab 4: Perulangan ‚Äì Melakukan Pekerjaan Berulang

4.1 While Loop

Selama kondisi benar, blok kode dijalankan terus.

# Menampilkan angka 1 sampai 5
i = 1
while i <= 5:
    print(i)
    i += 1   # jangan lupa increment, kalau tidak infinite loop

Penjelasan: i += 1 adalah cara singkat dari i = i + 1. Tanpa ini, nilai i akan tetap 1 dan loop berjalan selamanya.

4.2 For Loop dengan range()

Fungsi range() digunakan untuk menghasilkan urutan angka. Ada tiga bentuk umum:

  • range(stop) : menghasilkan angka dari 0 hingga stop-1. Contoh: range(5) ‚Üí 0,1,2,3,4
  • range(start, stop) : dari start hingga stop-1. range(2, 6) ‚Üí 2,3,4,5
  • range(start, stop, step) : dengan langkah step. range(1, 10, 2) ‚Üí 1,3,5,7,9

Contoh penggunaan:

for i in range(5):       # 0,1,2,3,4
    print(i)

for i in range(1, 6):    # 1,2,3,4,5
    print(i)

for i in range(2, 11, 2): # 2,4,6,8,10
    print(i)

Mengapa range() penting? Karena sering digunakan untuk mengulang sejumlah tertentu tanpa harus membuat variabel counter manual seperti while.

4.3 Break dan Continue

break menghentikan loop sepenuhnya. continue melompat ke iterasi berikutnya.

# Mencari angka pertama yang habis dibagi 7 antara 1-50
for i in range(1,51):
    if i % 7 == 0:
        print(f"Ketemu: {i}")
        break   # loop berhenti setelah menemukan

# Melewati angka genap
for i in range(1,11):
    if i % 2 == 0:
        continue
    print(i)   # hanya ganjil

Bab 5: Fungsi ‚Äì Membuat Blok Kode Sendiri

Fungsi adalah kumpulan kode yang diberi nama, dapat dipanggil berulang. Fungsi menghindari pengulangan kode dan membuat program lebih terstruktur.

5.1 Mendefinisikan Fungsi

Gunakan kata kunci def, diikuti nama fungsi, parameter dalam kurung (), dan diakhiri titik dua. Blok kode fungsi ditandai dengan indentasi.

def sapa(nama):
    """Fungsi untuk menyapa"""   # ini docstring (opsional)
    print(f"Halo {nama}!")

def luas_lingkaran(r):
    return 3.14 * r * r

sapa("Budi")
print(luas_lingkaran(7))

Penjelasan parameter: Parameter adalah variabel yang menerima nilai saat fungsi dipanggil. Dalam sapa(nama), nama adalah parameter.

5.2 Return vs Print

print() hanya menampilkan sesuatu ke layar, tapi tidak mengembalikan nilai ke pemanggil. return mengirimkan nilai balik, sehingga bisa disimpan dalam variabel atau digunakan dalam ekspresi.

def tambah(a, b):
    hasil = a + b
    return hasil   # mengembalikan nilai

x = tambah(3, 4)   # x akan bernilai 7
print(x)           # menampilkan 7

def cetak_tambah(a, b):
    print(a + b)   # hanya mencetak, tidak mengembalikan

y = cetak_tambah(3, 4)  # y akan bernilai None (karena tidak ada return)
print(y)                # None

Jadi, gunakan return jika kamu perlu menggunakan hasil perhitungan di tempat lain.

5.3 Parameter Default

Kita bisa memberi nilai default pada parameter, sehingga saat dipanggil bisa tanpa argumen.

def sapa_user(nama, pesan="Selamat belajar"):
    print(f"Halo {nama}, {pesan}!")

sapa_user("Dewi")                # menggunakan pesan default
sapa_user("Budi", "semangat ya") # mengganti pesan

Penjelasan: Parameter default harus diletakkan di akhir setelah parameter wajib.

5.4 Variabel Global dan Lokal

Variabel yang didefinisikan di luar fungsi bersifat global, bisa diakses di dalam fungsi. Namun jika ingin mengubah nilai global di dalam fungsi, perlu kata kunci global.

nilai_global = 10   # variabel global

def ubah_global():
    global nilai_global
    nilai_global = 20

print(nilai_global)   # 10
ubah_global()
print(nilai_global)   # 20

Jika tidak menggunakan global, maka Python akan membuat variabel lokal baru dengan nama yang sama, tidak mengubah global.

nilai = 5
def coba():
    nilai = 10   # ini lokal, tidak mengubah global
    print("dalam fungsi:", nilai)

coba()
print("luar fungsi:", nilai)   # tetap 5

Bab 6: Proyek Akhir ‚Äì Aplikasi Pengelolaan Nilai Siswa

Program ini akan menyimpan data siswa (nama, nilai matematika, ipa, inggris) dalam list of dictionaries. Fitur: tambah, tampilkan semua, cari, nilai tertinggi, dan simpan/load file.

6.1 List of Dictionaries

Kita akan menggunakan struktur data list of dictionaries. Setiap siswa adalah dictionary dengan key 'nama', 'mtk', 'ipa', 'ing'. Seluruhnya disimpan dalam list data_siswa.

data_siswa = []   # list kosong, nanti diisi dictionary

def tambah():
    n = input("Nama: ")
    mtk = float(input("Matematika: "))
    ipa = float(input("IPA: "))
    ing = float(input("Inggris: "))
    siswa = {'nama': n, 'mtk': mtk, 'ipa': ipa, 'ing': ing}
    data_siswa.append(siswa)
    print("Data tersimpan.")

6.2 Menampilkan Semua

def tampil():
    if not data_siswa:
        print("Belum ada data.")
        return
    for s in data_siswa:
        rata = (s['mtk'] + s['ipa'] + s['ing']) / 3
        print(f"{s['nama']} | MTK:{s['mtk']} IPA:{s['ipa']} ING:{s['ing']} | Rata: {rata:.2f}")

6.3 Mencari Siswa

def cari():
    nama = input("Nama yang dicari: ")
    for s in data_siswa:
        if s['nama'].lower() == nama.lower():
            rata = (s['mtk'] + s['ipa'] + s['ing']) / 3
            print(f"Ditemukan: {s['nama']} - Rata: {rata:.2f}")
            return
    print("Tidak ditemukan.")

6.4 Nilai Tertinggi

def tertinggi():
    if not data_siswa:
        print("Belum ada data.")
        return
    # Menggunakan max dengan key berupa fungsi lambda untuk menghitung rata-rata
    terbaik = max(data_siswa, key=lambda s: (s['mtk']+s['ipa']+s['ing'])/3)
    rata = (terbaik['mtk']+terbaik['ipa']+terbaik['ing'])/3
    print(f"Nilai tertinggi: {terbaik['nama']} dengan rata {rata:.2f}")

Penjelasan lambda: lambda s: (s['mtk']+...)/3 adalah fungsi anonim yang menerima s (siswa) dan mengembalikan rata-rata. Fungsi ini digunakan sebagai kriteria untuk mencari maksimum.

6.5 Menyimpan dan Memuat Data dengan File (with open)

Untuk menyimpan data agar tidak hilang saat program ditutup, kita gunakan file. Modul json memudahkan konversi list/dictionary ke teks.

Penjelasan with open: with open(nama_file, mode) as variabel: membuka file dan otomatis menutupnya setelah blok selesai, meskipun terjadi error. Ini lebih aman daripada open() dan close() manual.

import json

def simpan():
    with open("data_siswa.json", "w") as f:
        json.dump(data_siswa, f)
    print("Data tersimpan ke file.")

def muat():
    global data_siswa   # karena kita akan mengubah variabel global
    try:
        with open("data_siswa.json", "r") as f:
            data_siswa = json.load(f)
        print("Data dimuat dari file.")
    except FileNotFoundError:
        print("File tidak ditemukan. Mulai dengan data kosong.")

Detail:

  • "w" mode write (menulis, jika file ada akan ditimpa).
  • "r" mode read (membaca).
  • json.dump(obj, file) menulis objek ke file.
  • json.load(file) membaca objek dari file.
  • global data_siswa diperlukan karena kita mengassign nilai baru ke variabel global di dalam fungsi.

6.6 Program Utama dengan Menu

def menu():

Bab 7: Array (List) ‚Äì Menyimpan Banyak Data

List adalah struktur data yang dapat menyimpan banyak nilai dalam satu variabel. List bersifat mutable (dapat diubah).

7.1 Membuat List

List dibuat dengan tanda kurung siku []. Elemen dipisahkan koma.

buah = ["apel", "mangga", "jeruk"]
angka = [10, 20, 30]
campuran = ["teks", 123, True]

Mengapa pakai []? Karena itu adalah sintaks khusus untuk list. Python menggunakan [] untuk menunjukkan kumpulan data berurutan.

7.2 Mengakses Elemen

Indeks dimulai dari 0. Indeks negatif dari -1 (elemen terakhir).

buah[0]   # 'apel'
buah[-1]  # 'jeruk'

7.3 Operasi Dasar List

  • append(x) : tambah di akhir
  • insert(i, x) : sisip di indeks i
  • remove(x) : hapus elemen pertama yang bernilai x
  • pop(i) : hapus dan kembalikan elemen indeks i
  • index(x) : cari indeks pertama x
  • count(x) : hitung jumlah x
  • sort() : urutkan
  • reverse() : balik urutan
  • copy() : salin list

Perhatian: Menyalin dengan list2 = list1 tidak membuat salinan baru, hanya referensi. Gunakan list2 = list1.copy() atau list2 = list1[:].

7.4 List Comprehension

Cara ringkas membuat list baru berdasarkan list yang ada.

# Kuadrat bilangan 1-5
kuadrat = [x**2 for x in range(1,6)]   # [1,4,9,16,25]

# Hanya genap dari 1-10
genap = [x for x in range(1,11) if x % 2 == 0]   # [2,4,6,8,10]

7.5 Iterasi List

for b in buah:
    print(b)

for i in range(len(buah)):
    print(f"Indeks {i}: {buah[i]}")

for i, b in enumerate(buah):
    print(f"{i}: {b}")

enumerate menghasilkan pasangan (indeks, nilai).

Soal 7.7 ‚Äì Gabungan dengan Fungsi

Buat fungsi berikut:

  • rata_rata(list_angka) : mengembalikan rata-rata
  • nilai_maksimal(list_angka) : mengembalikan nilai maks (tanpa max bawaan)
  • nilai_minimal(list_angka) : mengembalikan nilai min (tanpa min bawaan)

Kemudian dalam program utama, minta user memasukkan 5 angka, panggil ketiga fungsi tersebut dan tampilkan hasilnya.

Jawaban:

def rata_rata(lst):
    return sum(lst) / len(lst) if lst else 0

def nilai_maksimal(lst):
    if not lst:
        return None
    maks = lst[0]
    for n in lst:
        if n > maks:
            maks = n
    return maks

def nilai_minimal(lst):
    if not lst:
        return None
    min = lst[0]
    for n in lst:
        if n < min:
            min = n
    return min

# Program utama

    n = float(input(f"Masukkan angka ke-{i+1}: "))
    angka.append(n)

print(f"Nilai maksimal: {nilai_maksimal(angka)}")
